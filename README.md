# Keep Track of Delicious Donuts
#### A donut tracking app using Flask and POSTGRESql.

## To Run App
In the app's main directory enter:
`flask run`

In you browser go to:
`localhost:5000`

## To Run Tests
In the app's main directory enter:
`python -m unittest tests -v`
